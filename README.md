# TV Show Search App

A simple and responsive TV Show Search App built using HTML, CSS, and JavaScript. The application uses the TVMaze API to search for TV shows and display detailed information including poster, release year, genre, rating, and summary.

## Features

* Search for any TV show
* Display show poster
* Display release year
* Display genres
* Display rating
* Display summary
* Error handling for invalid searches
* Loading indicator while fetching data
* Enter key support for quick searches
* Responsive card-based UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* TVMaze API

## API Used

TVMaze API

https://api.tvmaze.com/

## How It Works

1. User enters a TV show name.
2. The application sends a request to the TVMaze API.
3. The first matching result is retrieved.
4. Show details are displayed on the screen.
5. If no show is found, an error message is shown.
